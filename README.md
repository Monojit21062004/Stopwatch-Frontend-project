# Stopwatch-Frontend-project

A simple and responsive stopwatch application built using HTML, CSS, and JavaScript with start, stop, and reset functionality.

# Features

- Start, pause, and reset stopwatch

- Accurate time tracking (hours, minutes, seconds, milliseconds)

- Responsive and user-friendly UI

- Lightweight and fast performance

# Technologies Used

1. HTML5

2. CSS3

3. JavaScript

 # Future Enhancements

- Lap time functionality

- Dark/Light mode

- Keyboard shortcuts
